# School Management System

## Frontend
- Reactjs
- Tailwindcss
- Zustand (State Management)

**Frontend [live on](http://localhost:5173)**

### How to start frontend
```
npm install
npm run dev or npm start
```
## Backend
- Express
- mongoose
- jsonwebtoken
- bcryptjs

**Bacekend [live on](http://localhost:4000)**

```
npm install
npm run dev
```
## Routes
- ```/sign-in``` for login
- ```/sign-up``` for register
